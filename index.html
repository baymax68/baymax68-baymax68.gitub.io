Data Structures for Data Analysis for Data Visualization

I analyzed the distrabution of Dungeons and Dragons monsters Challenge level compared to their Name in alphabetical order. To do so I had to split the list into two halves each containing 80 monsters.

The Data

I recieved the data through an excel sheet which included: the name, challenge rating,challenge xp, and all of the ability scores and their modifiers. They were all in one collum seperated by ";" and in a random order. In order to fix this I seperated the data into collums by the ";". I think used the panda sort method to alphabetically sort all of the monsters names.

The Graph


After sorting and splitting the data set into two pieces I constructed the graph.

#
import pandas
import seaborn as sns
import matplotlib.pyplot as plt
def First():
    sns.set(rc = {'figure.figsize':(25,9)})
    plt.title = "name vs challenge rating"
    sns.set(font_scale=.5)
    plt.xlabel('Name', labelpad=40)
    sns.scatterplot(x='Name', y = 'ChallengeRating', data = first)
    plt.xticks(rotation=45)
    sns.despine
    plt.show()
    
def Second():
    sns.set(rc = {'figure.figsize':(25,9)})
    plt.title = "name vs challenge rating"
    sns.set(font_scale=.5)
    plt.xlabel('Name', labelpad=40)
    sns.scatterplot(x='Name', y = 'ChallengeRating', data = second)
    plt.xticks(rotation=45)
    sns.despine
    plt.show()
